You can read my blog posts on [http://JustinMaDev.github.io](http://JustinMaDev.github.io).

# How to run
Make sure you have installed `ruby`, `jekyll` and `bundle`
```
ruby -v
jekyll -v
bundle -v
```
Install them if not:
```
gem install jekyll bundler
```
Install the dependency:
```
bundle install
```
Then run:
```
jekyll serve
```
